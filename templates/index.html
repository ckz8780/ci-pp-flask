We can use for loops like this (index.html template and run.py):

    {% for number in list_of_numbers %}
  • {{ number }}
    • {% endfor %}

Or we can load data from a file and loop through it using keys/values (see run.py, data/data.json and index.html template):

    {% for item in json_data %}

  • Item:

      {% for key, value in item.items() %}
    • Key: {{ key }}, Value: {{ value }}
      • {% endfor %}
    • {% endfor %}

It's also possible to access keys/values by their names or positions (this prints out all the "name" keys):

    {% for item in json_data %}
  • Name: {{ item.name }}
    • {% endfor %}

We can access the loop index by using loop.index:

    {% for item in json_data %}
  • Loop Index: {{ loop.index }}
    • {% endfor %}

Or if you want it 0-based, you can use loop.index0:

    {% for item in json_data %}
  • Loop Index: {{ loop.index0 }}
    • {% endfor %}

Finally, we can use if-then logic (this prints out only key/value pairs if the value contains the string '2'):

    {% for item in json_data %} {% for key, value in item.items() %} {% if '2' in value %}
  • Key: {{ key }}, Value: {{ value }}
    • {% endif %} {% endfor %} {% endfor %}

Let's put it together and use some advanced routing:

    {% for item in json_data %}
  • {{ item['name']}}
    • {% endfor %}
